# KOTDEV

Персональный бренд и сайт-портфолио Виталия Кота — Senior Unity Developer.

## О проекте

Сайт создан для представления Виталия как профессионального Unity-разработчика. Содержит информацию об опыте, навыках, портфолио и контакты.

Поддерживается два языка: русский и английский.

## Как запустить

1. Клонировать репозиторий или скачать архив
2. Открыть index.html в браузере
